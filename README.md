# Vite React with React Router & React Bootstrap

A modern React application scaffolded with Vite, styled using Bootstrap and React Bootstrap powered by essential developer tools for learning and scalability.

If you’re new to React and frustrated with setup, this template gives you a clean, guided, and reliable way to start building modern apps quickly!!

---

## 🚀 Tech Stack

- **React 19.2.3**
- **React DOM 19.2.3**
- **React Router DOM 7.9.1**
- **Vite 7.3.0**
- **@vitejs/plugin-react 5.1.2**
- **React Bootstrap 2.10.10**
- **Bootstrap 5.3.8**
- **TypeScript typings for React (19.2.8) & React DOM (19.2.3)**
- **PropTypes 15.8.1**
- **ESLint 9.39.1 & @eslint/js 9.39.1**
- **eslint-plugin-react-hooks 7.0.1**
- **eslint-plugin-react-refresh 0.4.26**
- **globals 17.0.0**
- **Utilities:** LocalForage 1.10.0, Match-sorter 8.1.0, Sort-by 1.2.0
- **UI Enhancements:** React Icons 5.5.0, React Toastify 11.0.5
